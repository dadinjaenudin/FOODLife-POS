About Terminal Management

Terminal Status: A terminal is considered Online (●) if it sent a heartbeat within the last 5 minutes. If no heartbeat is received for more than 5 minutes, it's marked as Offline (●). Inactive terminals (▐▐) are deliberately deactivated and won't accept new transactions.

Device Types:

  • POS / Kasir: Main cashier terminal for processing orders and payments
  • Tablet / Waiter: Handheld device for waiters to take orders
  • Self-Service Kiosk: Customer-facing ordering kiosk
  • Kitchen Display: Display screen for kitchen to view orders

Features: Each terminal can be configured with different payment methods, printer settings, and hardware integrations. Use the Duplicate feature to quickly set up multiple terminals with similar configurations.
